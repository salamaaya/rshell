### Build
```bash
cargo build
```

### Run
```bash
cargo run
```

### Test
```bash
cargo test 
```

### Example Usage 
```bash
$ cargo run
rshell$ ls
Cargo.lock  Cargo.toml  README.md  REQUIREMENTS.md  src  target
rshell$ exit
$
```
